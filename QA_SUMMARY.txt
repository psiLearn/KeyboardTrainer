### 🔴 Issue #5: Backspace Logic Bug
```
File: src/Client/Pages/TypingView.fs (line 109)

Problem: String slicing fails at edge cases

Current Code:
    | Backspace ->
        if model.TypingState = InProgress && model.CurrentCharIndex > 0 then
            let newInput = model.UserInput.[0..model.CurrentCharIndex - 2]

Example Failure:
    CurrentCharIndex = 1
    UserInput = "a"
    newInput = "a".[0..(-1)]    ← StringIndexOutOfRangeException!

Impact: Runtime crash when user presses backspace after typing 1 character
```
