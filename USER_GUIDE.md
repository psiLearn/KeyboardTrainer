# Keyboard Trainer User Guide

## Overview

Keyboard Trainer helps you practice typing speed and accuracy using short lessons.
You can track your progress and view metrics over time.

## Start a Lesson

1. Open the app in your browser.
2. Pick a lesson from the list on the Start page.
3. Selecting a lesson opens the typing view automatically.
4. Click **Start Typing** (or press `Enter`/`Space`) to begin.

## Training Aids

On the Start page you can enable optional training aids:

- **Color letters**: keeps correct/incorrect highlights.
- **Show keyboard**: displays an on-screen keyboard while typing.
- **Highlight next key**: highlights the next required key.

These settings are saved in your browser and persist across reloads.

### Keyboard Shortcuts

- Start typing: `Enter` or `Space`
- Cancel typing: `Esc`
- Submit results (after completion): `Enter`
- Reset after completion: `R` or `Esc`

## Typing Experience

- Characters are highlighted as you type.
- Errors are shown in red.
- Metrics are calculated in real-time.
- If enabled, the on-screen keyboard highlights the next key and the last key you pressed (correct/error).

When you finish the text, click **Submit Results** to save your session.

## Statistics

Open **Statistics** from the top navigation to view:

- Average WPM / CPM / Accuracy
- Recent sessions
- Pending local sessions (if you typed while offline)

## Offline Mode

If you go offline while submitting:

- The session is stored locally.
- A pending count appears on the Start and Statistics screens.
- The app retries syncing when you are online again.

You can clear local data from the Statistics page if needed.

## Troubleshooting

- If lessons do not load, use the **Retry** button.
- If you see an error banner, verify the server is running and reachable.
- If the UI looks empty after a deploy, do a hard refresh in the browser.
